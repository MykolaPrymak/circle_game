# HTML5 Circle game #

This is a simple [game](http://mykolaprymak.github.io/circle_game/) build with canvas and audio HTML5 features.


## Rules ##
* move your circle by mouse
* eat smaller circle and grow
* dont die :)

* [Browser Quest](http://browserquest.mozilla.org/)
